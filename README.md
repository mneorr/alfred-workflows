## Alfred workflows

Here's the set of workflows I'm using as global hotkeys to bring up apps.

* Open up Sublime Text:
```
Ctrl + CMD + S
```

* Open up browser:
```
Ctrl + CMD + B
```

* Now you need Terminal to commit changes
```
Alt + Esc
```

* Check your email:
```
Ctrl + CMD + M
```

* Read Twitter:
```
Ctrl + CMD + T
```

* Open up Skype:
```
Ctrl + CMD + K
```
